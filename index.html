Open-source software — it's good for you!

Mission Statement

HaskAll is an open-source software collective. Our aim is to build software that supports the projects of social enterprises worldwide. We write everything in Haskell, a programming language with its roots in research and open-source. We are open to project suggestions, from rebuilding proprietary software for an open-source era, to blue-sky ideas for a techno-utopian future.

Suggestions

We are currently working on a web application to collect and manage open-source project suggestions from the community. If you have a suggestion we would love to hear from you — for now please open an issue on GitHub here. Give your suggestion a title, a nice description, and label it on the right-hand side with "Project Suggestion". If you don't want to create a GitHub account, you can email your suggestion to us here.
